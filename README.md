# MDS API 42

API Documentation : 
https://api4210.docs.apiary.io/#

* Default server port : `3031`
* Default uri : `http://localhost:3031`

## Run mongo database
```
mongod --port {db_port} --dbpath '{db_path}'
```
## Run pre-configured connection
```
npm run startmongo
```

## Install & Run 

```
npm run build

```

## Run 

```
npm run start

```

## Test

```
npm run test

```
